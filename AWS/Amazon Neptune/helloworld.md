Amazon Neptune은 SPARQL과 Gremlin을 모두 사용할 수 있는 그래프 데이터베이스 엔진입니다. SPARQL은 그래프 데이터에 액세스하고 상호 작용하는 데 사용되며, Gremlin은 그래프를 쿼리하는 데 사용됩니다. 
SPARQL
W3C의 쿼리 언어로, RDF 데이터를 포함하는 그래프를 쿼리하는 데 사용됩니다 
Amazon Neptune 사용 설명서에서 SPARQL 쿼리 언어를 사용하는 방법을 확인할 수 있습니다 
SPARQL explain 기능을 사용하면 Neptune 엔진이 실행하는 접근 방식을 이해할 수 있습니다 
Gremlin
Apache TinkerPop의 순회 언어로, 속성 그래프를 쿼리하는 데 사용됩니다 
Amazon AWS Documentation에서 Gremlin API를 사용하는 방법을 확인할 수 있습니다
